# MERN Job Portal

A full-stack job portal web application built with the MERN stack.

## 🔧 Tech Stack
- **MongoDB**
- **Express.js**
- **React.js**
- **Node.js**

## 💼 Features
- JWT-based user authentication
- Role-based dashboards (Recruiter / Applicant)
- Job posting & application tracking
- RESTful API integration
